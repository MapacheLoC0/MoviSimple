Algoritmo de Dijkstra

El algoritmo de Dijkstra es un algoritmo para encontrar el camino más corto entre nodos en un grafo con pesos no negativos.

Pasos del algoritmo:

  1. Inicializar distancias de todos los nodos a infinito, excepto el nodo origen (distancia = 0)
  2. Marcar todos los nodos como no visitados
  3. Mientras haya nodos no visitados:
  4. Seleccionar el nodo no visitado con menor distancia
  5. Marcar el nodo como visitado
  6. Para cada vecino no visitado, calcular nueva distancia y actualizar si es menor
